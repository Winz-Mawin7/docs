›คู่มือเกี่ยวกับวิชาเรียน

คู่มือเกี่ยวกับเรื่องทั่วไป

  • การเข้าสู่ระบบ
  • คู่มือการตั้งค่าสำหรับผู้ดูแลระบบ

คู่มือเกี่ยวกับข้อมูลนักเรียน

  • การเพิ่มข้อมูลนักเรียนเข้าใหม่
  • การบ้าน
  • การจัดการคะแนนความประพฤติ
  • บันทึกข้อมูลสุขภาพ
  • แก้ไขสถานะนักเรียน

คู่มือเกี่ยวกับการลงเวลา

  • การยืม-คืนบัตรสำรอง
  • บันทึกการลงเวลาของนักเรียนแบบ Manual และ แบบกลุ่ม
  • บันทึกการลงเวลาของบุคลากรแบบ Manual
  • บันทึกการลาของนักเรียนแบบรายบุคคล และ การลาแบบรายกลุ่ม
  • บันทึกการลาของครู
  • สร้างกิจกรรมโรงเรียน

คู่มือเกี่ยวกับการแจ้งข่าวสาร

  • แจ้งข่าวสาร GPA
  • การแจ้งข่าวสารแบบกลุ่ม แบบเดี่ยว แบบทั้งโรงเรียน

คู่มือเกี่ยวกับวิชาเรียน

  • ตารางสอน
  • จัดตาราง(นักเรียน)
  • จัดตารางสอน(ครู)
  • การลาระหว่างวัน
  • การเช็คชื่อนักเรียนรายคาบรายวัน
  • การยกเลิกการสอน
  • คู่มือการตั้งค่าตารางสอนสำหรับ Admin

คู่มือการประเมินพฤติกรรม

  • สถานะการประเมิน SDQ
  • ประเมินด่วน SDQ
  • นักเรียนมีความเสี่ยง SDQ
  • สถานะการประเมิน EQ
  • การประเมินด่วน EQ

คู่มือการรับสมัครนักเรียนและโปรแกรมช่วยจัดตารางสอน

  • การจัดตารางสอน
  • การเปิดรับสมัครการสอบ Admission
  • การเปิดรับสมัครวิชาเลือก

คู่มือการใช้ห้องพยาบาล

  • การใช้งานระบบห้องพยาบาล
  • การใช้งานระบบห้องสมุด

คู่มือการสั่งบัตรใหม่ผ่าน store

  • การสั่งซื้อบัตรใหม่ทาง Store

คู่มือเกี่ยวกับระบบศูนย์อาหาร

  • คู่มือการใช้งานระบบศูนย์อาหาร

คู่มือเกี่ยวกับครื่องสแกนบัตรและ Hardware ต่าง ๆ

  • สถานะของไฟที่ตัวเครื่องสแกนบัตร
  • คู่มือการใช้งานเกี่ยวกับเครื่องศูนย์อาหาร
  • เครื่องเช็คชื่อรายวิชา SIC

คู่มือการดาวน์โหลดและใช้งาน Application Nextschool

  • การใช้งาน Application Nextschool
  • การใช้งาน Application Nextschool (สำหรับผู้ปกครอง)
  • การใช้งาน Application Nextschool (นักเรียน)

คู่มือการใช้งานเว็บ Student.nextschool

  • การใช้งานระบบ Student.nextschool สำหรับนักเรียน

ตารางสอน

เมนูวิชาเรียน

1. ตารางสอน

  • วิชาเรียน->ตารางสอน
  • เมื่อคลิ๊กที่วิชาสอนแล้วจะเจอ รายละเอียด Section ชั้นเรียนที่สอน

2. นักเรียน

  • การเช็คชื่อผ่านเครื่องสแกนรายวิชา (กรณีนักเรียนไม่ได้สแกนบัตรหรือไม่ได้นำบัตรมา ครูผู้สอนสามารถมาที่แก้ไขเปิดโหมดการลงเวลาได้)
  • เมื่อเปิดโหมดการลงเวลาแล้วให้คลิ๊กที่ช่อง สี่เหลี่ยมเล็กๆเมื่อคลิ๊กจากเครื่องหมายกากบาทจะเป็นเครื่องหมายถูก (กรณีที่นักเรียนทำการแตะบัตรที่เครื่องสแกนรายวิชาจะขึ้นเป็นเครื่องหมายกุญแจไม่สามารถแก้ไขได้)
  • เมื่อแก้ไขเรียบร้อยแล้วให้กดปิดโหมดแก้ไข
  • มีรายงาน ป.พ.5 การเข้าเรียน (PDF)

หมายเหตุ รายงาน ป.พ.5 การเข้าเรียน มาจากข้อมูลพื้นฐานการลงเวลาผ่านเครื่องสแกนเคลื่อนที่ในแต่ละรายวิชา

3. ยกเลิกการสอน

  • คลิ๊กที่ สร้างการยกเลิกสอน
  • เลือกวิชาที่ครูสอน ช่วงเวลาเรียน พร้อมเหตุผล ที่ต้องการยกเลิกการสอน และ กดปุ่ม เพิ่ม

4. บันทึกสอนแทน

  • คุณสามารถสอนแทนได้เฉพาะวิชาในกลุ่มสาระที่คุณมีตารางสอนเท่านั้น
  • หากวันและเวลาสอน ไม่ตรงกับเวลาที่สอนแทน ให้เลือกวันและเวลาสอนแทนใหม่หลังจากเลือก Section และวิชาแล้ว
  • หากเป็นการบันทึกสำหรับเปลี่ยนวันเวลาเรียน วันและเวลาจะต้องไม่ตรงกับวันเวลาปกติ

5. บันทึกคะแนน

  • สามารถบันทึกคะแนน กลางภาค ก่อนกลางภาค หลังกลางภาค ปลายภาค
  • เมื่อใส่คะแนนแล้วระบบจะบันทึกให้อัตโนมัติ
  • มีรายงาน ป.พ.5 คะแนน (PDF)

หมายเหตุ รายงาน ป.พ.5 คะแนน มาจากข้อมูลพื้นฐานที่ครูทำการบันทึกคะแนนในแต่ละ ภาคเรียน

  • : ก่อนจะบันทึกคะแนน ต้องสร้างรายงานบันทึกคะแนน Section ก่อนโดย Admin จะสร้างผ่านการตั้งค่าในวิชาเรียน
